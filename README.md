# 🏥 AI-Powered Medical Image Analysis

## 📌 Overview
This project uses Deep Learning (CNN) to analyze medical images like X-rays and detect diseases such as pneumonia.

## 🧠 Features
- Image preprocessing
- CNN model for disease detection
- Accuracy evaluation
- Real-time prediction system

## 🛠️ Tech Stack
Python, TensorFlow, OpenCV, NumPy

## ▶️ Run Project
pip install -r requirements.txt
python src/main.py

## 📂 Dataset
https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia
